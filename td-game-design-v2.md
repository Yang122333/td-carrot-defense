# 🥕 《萝卜保卫战》— H5 塔防小游戏 v2.0 策划补充案

> 版本：v2.0 增量更新 | 策划牛 📋 出品 | 2026-03-12
> 
> 本文档为 v1.0 的增量补充，旨在提升游戏的**视觉表现力**、**策略深度**与**整体精致感**。

---

## 1. 地图自然化：菜园生态升级

为了打破机械的网格感，我们将 S 型路径进行「柔化」处理，并增加环境细节。

### 1.1 路径形态优化（曲线化）
- **核心逻辑**：路径不再严格贴合格子中心，而是采用 **Bézier 曲线** 或 **多段弧线** 连接关键拐点。
- **视觉表现**：
  - 路径宽度保持 48px，但边缘增加随机的「毛边」效果（碎石、杂草）。
  - 在拐弯处增加「泥土堆」和「车轮印」装饰。
  - **路径颜色**：从纯 #F5DEB3 变为带有深浅斑驳感的 #D2B48C (褐赭色) 到 #F5DEB3 的渐变。

### 1.2 环境装饰分布表
| 装饰物 | 视觉特征 | 放置规则 | 互动效果 |
| :--- | :--- | :--- | :--- |
| **栅栏** | 棕色木质，倾斜角度 | 随机分布在路径转角外侧 | P2: 怪物经过可能撞歪 |
| **动态水塘** | 半透明天蓝色，带白边波纹 | 占据 2x2 格子（不可放塔区） | 增加画面灵动感 |
| **小花/杂草** | 红/黄/白小圆点，带绿色短线 | 路径两侧 1 格内随机散布 | 随风微微晃动（Sin 偏移） |
| **路标** | 木牌写着 "Stay Away!" | 入口处 | 鼠标悬停会抖动 |
| **蝴蝶** | 两个彩色三角组成 | 在花丛上方随机路径飞行 | 增加生机，纯装饰 |

### 1.3 草丛层次感
- **底层**：深绿色背景 (#228B22)。
- **中层**：大块的浅绿色草皮 (#90EE90)，边缘羽化。
- **表层**：在可放塔格子上增加「修剪整齐」的亮绿色框感，提示玩家此处可交互。

## 2. 新功能设计（体验提升核心）

经过评估，我选择了以下 4 个对 Demo 体验提升最大的功能：

### 2.1 【P0】塔的攻击优先级切换（策略核心）
玩家点击已放置的塔时，在升级/出售按钮旁增加「目标策略」切换图标。
- **默认 (First)**：攻击距离萝卜最近的敌人。
- **最强 (Strongest)**：优先攻击当前生命值最高的敌人（针对铁甲虫）。
- **最弱 (Weakest)**：优先攻击生命值最低的敌人（补刀）。
- **最后 (Last)**：攻击离入口最近的敌人。

### 2.2 【P1】金币掉落与吸收动画（爽感核心）
- **机制**：怪物死亡时，金币不再是简单的数字飘字，而是生成一个旋转的 **金色硬币粒子**。
- **动画逻辑**：
  1. 金币弹出（带有随机的向上初速度和水平偏移）。
  2. 停顿 0.2 秒。
  3. 快速飞向左上角的 HUD 金币余额处。
  4. 余额数字产生「放大缩小」的缩放反馈（Juice Effects）。

### 2.3 【P1】怪物路径预览（易用性核心）
- **触发**：在准备波次的 8 秒期间，或鼠标悬停在「下一波」按钮上时。
- **表现**：在路径上显示半透明的「足迹粒子」或「虚线箭头」，从入口缓缓流向终点。
- **价值**：明确告知玩家下一波怪物的行进路线（尤其是多路径扩展时非常重要）。

### 2.4 【P2】第 5 种炮塔：🔥 烈焰向日葵（强力辅助）
| 属性 | 数值 | 逻辑 |
| :--- | :--- | :--- |
| **名称** | 烈焰向日葵 | 负责给周围炮塔「上 Buff」 |
| **外观** | 橙黄色的向日葵，戴着火焰护目镜 | 360 度光环 |
| **核心机制** | **光环增益**：使射程内所有己方塔伤害 +20% | 不直接攻击 |
| **造价** | 250 💰 | 高级策略单位 |
| **升级 (Lv2)** | 造价 200💰 | 额外提供 15% 攻速加成 |

## 3. 视觉体验提升（精致感细节）

### 3.1 分层背景描述
- **层 1（远景）**：模糊的森林剪影，天边有淡淡的云朵。
- **层 2（中景）**：菜园的木质围栏，远处的谷仓缩略图。
- **层 3（近景）**：棋盘状草地与路径，带有柔和的阴影（向右下偏移 4px）。

### 3.2 丰富的粒子特效清单
- **玉米射手**：命中时溅射出「黄色碎屑」。
- **冰豌豆**：命中时产生「深蓝色冻结光环」爆发，怪物身上持续掉落「白色雪花」。
- **辣椒大炮**：炮口喷出「烟雾粒子」，爆炸中心产生「火星升腾」。
- **萝卜被吃**：萝卜产生「受击抖动」，掉落 1-2 片「橙色碎片」。

### 3.3 UI 美化方案
- **字体**：使用圆润的无衬线字体（如 `Arial Rounded MT Bold` 或自定义 Canvas 描边字）。
- **动效**：按钮在 Hover 时变大 1.1 倍，点击时下沉。
- **血条设计**：怪物血条改为「胶囊型」，背景半透明黑，血条颜色根据百分比由绿变红。

## 4. 补充数值平衡调整

为了适应新功能，对现有数值做微调：
- **铁甲虫**：血量由 200 下调至 **180**，但基础移动速度由 0.6 提升至 **0.7**（配合优先级切换系统，让玩家更需要操作）。
- **冰豌豆**：减速效果对 Boss（虫王）的持续时间减半（改为 1 秒），避免 Boss 被风筝到死。

## 5. 【紧急更新】难度与平衡性大调（Hardcore Update）

根据 Van 哥反馈，当前版本过于“割草”。为了提升策略性，我们将进行**经济紧缩**与**怪物数值重构**。

### 5.1 核心经济与生命值削减
| 参数 | 原数值 | **调整后数值** | 调整目的 |
| :--- | :--- | :--- | :--- |
| **初始金币** | 300 💰 | **180 💰** | 开局只能放 1 个强力塔或 1 塔 1 地雷，杜绝开局铺场。 |
| **萝卜生命值** | 20 ❤️ | **10 ❤️** | 容错率降低 50%，漏怪惩罚更重，增加紧迫感。 |
| **波次奖励** | 50 💰 | **30 💰** | 降低固定收入，逼迫玩家依赖击杀奖励和跳过奖励。 |
| **小青虫击杀** | 10 💰 | **8 💰** | 微调前期收益，拉长阵线建设周期。 |

### 5.2 塔防单位性价比平衡
| 塔名称 | 调整点 | 目的 |
| :--- | :--- | :--- |
| **玉米射手** | 伤害 15 → **12** | 降低基础单体性价比，不能靠纯堆玉米过关。 |
| **冰豌豆** | 造价 150 → **180** | 增加控制成本，控制塔属于高级策略，不应廉价。 |
| **辣椒大炮** | 攻速 2.0 → **2.5s** | 显著拉长装填时间，增加其“AOE 爆发”的空窗期风险。 |
| **蘑菇地雷** | 造价 80 → **100** | 提升一次性防御的成本。 |

### 5.3 怪物强化与混编波次表（8 波重排）
*核心逻辑：缩短出怪间隔，增加混编比例，制造“速度”与“血量”的双重压力。*

| 波次 | 怪物组合 (数量) | 出怪间隔 | 难度评分 | 战术要点 |
| :--- | :--- | :--- | :--- | :--- |
| **W1** | 小青虫 x6 | 1.2s | ⭐ | 必须补足输出，否则 1 个玉米射手打不完 6 只。 |
| **W2** | 小青虫 x5 + 红蚂蚁 x2 | **0.8s** | ⭐⭐ | 快速怪穿插，考验塔的目标锁定。 |
| **W3** | 红蚂蚁 x6 | 0.6s | ⭐⭐⭐ | **第一道坎**。纯速通压力，必须有冰豌豆或 2 个以上玉米。 |

### 5.4 怪物属性微调（抗性引入）
- **铁甲虫**：血量 180 (不变)，但**移动速度在血量低于 50% 时会狂暴**（提升至 1.0）。
- **红蚂蚁**：增加 10% 闪避率（炮弹有概率落空，需通过粒子效果反馈）。
- **虫王 Boss**：血量上调至 **800**，且免疫 1.5s 以上的硬控，减速上限固定在 30%。
