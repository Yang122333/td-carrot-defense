# 🏰 《萝卜保卫战》— H5 塔防小游戏 Demo 策划案

> 版本：Demo v0.1 | 策划牛 📋 出品 | 2026-03-11

---

## 1. 游戏概述

### 一句话核心玩法
**在菜园里建造蔬菜炮塔，阻止贪吃虫子偷走你的超级大萝卜！**

### 世界观 / 主题
在一个 Q 版卡通菜园里，玩家是菜园的守护者。菜园深处长着一颗传说中的「黄金大萝卜」🥕，各种贪吃虫子闻香而来，沿着田间小路想要偷走它。玩家需要种下各种蔬菜战士来保卫萝卜！

**风格关键词**：圆润可爱、色彩鲜艳、表情夸张、萌系卡通

---

## 2. 核心循环

### 玩家每局在做什么
```
观察路径 → 选择炮塔 → 放置到格子 → 开波 → 观察战斗 → 赚金币 → 升级/补��� → 下一波
```

### 正反馈设计
| 反馈点 | 表现 |
|--------|------|
| 击杀怪物 | 怪物 Q 萌爆炸动画 + 金币飞出 + 「+10」飘字 |
| 购买/放置塔 | 塔从土里「蹦」出来的弹跳动画 |
| 升级塔 | 闪光特效 + 塔变大一圈 + 表情变凶 |
| 清完一波 | 「Wave Clear!」横幅 + 奖励金币雨 |
| 通关胜利 | 大萝卜开心跳舞 + 烟花 🎆 |

## 7. 经济系统

### 金币参数

| 参数 | 数值 |
|------|------|
| 初始金币 | 300 💰 |
| 每波基础奖励 | 50 💰（完成该波后发放） |
| 跳过准备时间奖励 | 20 💰 |
| 小青虫击杀 | 10 💰 |
| 红蚂蚁击杀 | 15 💰 |
| 铁甲虫击杀 | 25 💰 |
| 虫王击杀 | 100 💰 |

### 经济验算

| 时间点 | 预期累计金币 | 可购买 |
|--------|-------------|--------|
| 开局 | 300 | 1 个玉米射手(100) + 1 个冰豌豆(150)，或 3 个玉米射手 |
| 第 1 波后 | 300 + 50 + 50 = 400（剩余） | 可再买 1-2 塔 |
| 第 3 波后 | 约 600-700 | 可买辣椒大炮 + 升级 |
| 第 5 波后 | 约 900-1100 | 需要策略分配升级 vs 新塔 |
| 第 8 波（Boss） | 约 1200-1500 | 全力输出配置 |

> 设计原则：前 2 波保证玩家至少能放 2-3 个塔；中期需要在「升级」和「新建」之间做选择；Boss 波需要合理布阵才能过。

### 技术要点
- **渲染**：单个 `<canvas>` 元素，`requestAnimationFrame` 驱动
- **输入**：鼠标点击 + 触屏 touch 事件（兼容移动端）
- **碰撞**：圆形碰撞检测（距离 < 半径之和）
- **状态管理**：简单状态机 `{ MENU, PLAYING, WIN, LOSE }`
- **数据驱动**：塔/怪/波次配置用 JS 对象字面量，方便调参

### 出售规则
- 出售返还：塔造价的 **50%**（含升级花费）
